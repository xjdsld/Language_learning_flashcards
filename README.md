# Language_learning_flashcards

Консольна версія застосунку надаватиме можливість користувачу повторювати та вивчати нові слова. Користувач зможе додавати слово українською, а переклад має підтягуватись через АПІ сервіс. Таким чином застосунок працюватиме по типу флеш-карток, де користувачу виводиметься слово, він згадуватиме його переклад, а потім, вписуватиме обрану літеру чи номер, що означатиме "перевірити". Після цього користувачу буде показано правильний варіант.
Також буде сожливість обрати кількість карток в сесії зі стандартних опцій. Ця, як і інші функції, реадізовуватимуться через класи, а увесь проект підпорядковуватиметься принципам ООП. 
Проєкт складатиметься з окремих модулів: один модуль відповідатиме за БД, інший за взаємодівю з АПІ сервісом, за сесію, а інший за загальне функціонування. 

ПРриблизно структура буде наступною:
project_folder:
- main.py
  
- API.py
  
- DB.py
  
-session.py

Опціонально, за можливості, якщо буде в АПІ сервісі, додаватиметься варіант речення з правильним вживанням слова. Також можливо буде вбудований режим з таймером, який через певну кількість секунд показуватиме правильну відповідь.
Загалом такий проект буде спрямований для допомоги у навчанні, у цьому конкретному варіанті вивченні англійської.
